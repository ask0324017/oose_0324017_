#題目：老師評價系統

## 工作分配
## 一、文字說明：

是一個可以給學生查詢與編輯老師評價的系統，以學校信箱註冊帳號，就能以名子查詢老師資訊，或是查詢老師開的課程的評價，也能給予老師評價與分數，讓學生有更多資訊做參考與選擇。
